yFiles for HTML Tools

This directory contains tools that simplify development with yFiles for HTML. In particular, they can support you with the following tasks

demos-static-build

Contains a Vite config that can be used to build a stand-alone, static version of the demos. This can be used to deploy the demos to an internal static web server.

es-modules-to-umd, umd-to-es-modules

Converts source code from ES module imports to UMD-style imports, and from UMD-style imports to ES module imports.

transform-split-import

Converts a single import { â€¦ } from 'yfiles' statement to import statements that use the specific yFiles module where the imported types reside.

unpack-library-files

Extracts the yFiles for HTML library .tgz files located in the lib and lib-dev directories.

vite-plugin-runtime-error

A Vite plugin for the demo dev server, that displays runtime errors in the vite-error-overlay with clickable file locations.

You can find more detailed information about each tool in the Developer's Guide Utilities section.
